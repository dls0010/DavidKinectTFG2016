Home screen

In the first screen we are at the beginning of session, you have two options::

  • 1. If you are not registered in the system, should press REGISTER.
  • 2. If you are already registered, fill in the username and password and press fields LOGIN.

Pantalla registro

In the registration screen, we have a form with the following fields:

  • User: username
  • Password and repeat password for the user account.
  • User type: user type that we want to register. (Patient Therapist Administrator).

  • Patient registration:

    In this window you have to fill in the personal data of the patient, we can even add a picture to associate the patient, we can get that picture, as a file in your computer, or through a photograph Kinect by the camera. Finally you can add a special description about the patient, if you have problems need to know the therapist so that the allocation exercise does not create problems.

  • Therapist registration:

    In this window you have to fill in personal data therapist, we can even add a picture to associate the therapist, we can get that picture, as a file in your computer, or through a photograph by the Kinect camera

  • Administrator registration:

    In this window you have to fill in personal data administrator.

Logins

The login screen will vary depending on the type of user:

    • Patient home screen:

      • Select an exercise from the list and do it.
      • Check my exercises to date.
      • Check my workouts assigned by the therapist.

    • Therapist home screen:

      • Adquerir of patients who are in a free state to your inquiry.
      • Check that patients are assigned this therapist.
      • Create a workout with a series of exercises the patient to choose.
      • View trainings conducted by their patients.

    • Administrator home screen:

      • Change patients chart database.
      • Change table therapists database.
